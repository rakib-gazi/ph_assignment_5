What is the Document Object Model (DOM)?

The definition of DOM according to me is given below :

The DOM stands for Document Object Model. The Document refers to the all elements which is situated inside the Html tag or elements. we can easily triggered any elements & manipulated it using Javascript DOM Method. So DOM is interesting feature of Javascript.

How do you select an element from the DOM?

We can select elements form DOM in most common five way, Such as.

  • document.getElementById(): This method selects an element by its unique id.
  • document.getElementsByClassName(): This selects all elements with a specified class name.
  • document.getElementsByTagName(): This method selects elements by their tag name (e.g., div, p).
  • document.querySelector(): This method selects the first element that matches a given CSS selector.
  • document.querySelectorAll(): Similar to querySelector(), this method returns all elements matching a CSS selector.
What is event delegation in the context of the DOM, and why is it useful?

How Event Delegation Works:

Event Delegation is pattern which used in parent DOM, where an Event Listener added in parent then access all child using parent Event Listener.

Why Event Delegation is Useful:

  • Boosting Performance: when have many child element then we can use one event listener instead of using individuals child event listener
  • Dynamic Elements: we can add dynamically child element using parent event listener.
  • Simplified Code: Its Simplified our code which increase more readability of code.
How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles using the DOM in various way. Some of Example are given below.

Attribute Manipulate:

  • getAttribute(): Retrieves the value of an attribute.
  • setAttribute(): Sets a new value for an attribute.
  • removeAttribute(): Removes an attribute.

Style Manipulate:

  • element.style.property = value: The style property is used to set or modify the inline styles of an element. In JavaScript, CSS property names are written in camelCase.
  • element.style.cssText: This property can be used to set multiple CSS styles at once in a string format.

    • For non-inline styles, we can use the getComputedStyle() method to get the final computed values of styles applied to an element, which includes styles from stylesheets and inline styles.

Using classList for Style Manipulation:

  • add(className): Adds a specified class to the element.
  • remove(className): Removes a specified class from the element.
  • toggle(className): Toggles the presence of a specified class on the element.
  • contains(className): Checks whether the element has a specified class.
